# 사용법:eyes:

### 설치

config 폴더 안에 dev.js 생성 후  
자신이 가입되어 있는 mongoDB의 클러스터 정보를 입력  
루트 및 클라이언트 경로에서 yarn install

### 실행

루트 경로에서 yarn dev
